매일읽음 개인정보처리방침

최종 수정일: 2026년 5월 1일 · 시행일: 2026년 5월 1일

매일읽음(Daily Read)은 이용자의 개인정보를 어떠한 형태로도 수집하지 않습니다. 모든 데이터는 이용자의 기기 내부에만 저장됩니다.

1. 개인정보 수집 여부

매일읽음은 이용자의 개인정보를 일체 수집하지 않습니다. 본 앱은 별도의 회원가입이나 로그인 절차가 없으며, 이용자의 이름, 이메일, 전화번호, 위치, 연락처 등 어떠한 식별 가능한 정보도 요구하지 않습니다.

2. 데이터 저장 방식

이용자가 앱에 입력하는 모든 데이터는 오직 이용자 본인의 기기 내부 저장소(SwiftData)에만 저장됩니다.

저장되는 데이터 저장 위치 외부 전송 여부
도서 정보(제목, 저자, 출판사 등) 기기 내부 전송하지 않음
독서 기록(평점, 한줄평, 리뷰) 기기 내부 전송하지 않음
독서 세션(타이머 기록) 기기 내부 전송하지 않음
글귀 및 메모 기기 내부 전송하지 않음
책 표지 이미지 기기 내부 전송하지 않음
독서 목표 설정값 기기 내부 전송하지 않음

3. 권한 사용

매일읽음은 다음의 기기 권한을 요청할 수 있으며, 각 권한의 사용 목적은 아래와 같습니다.

권한 사용 목적 거부 시 영향
알림(Notifications) 일일 독서 리마인더, 글귀 랜덤 알림 알림 기능 사용 불가, 다른 기능은 정상 사용 가능
사진(Photos) 책 표지 이미지를 갤러리에서 선택 책 표지 등록 불가, 다른 기능은 정상 사용 가능
카메라(Camera) 책 표지를 직접 촬영하여 등록 (선택 사항) 카메라로 표지 촬영 불가, 갤러리 선택은 가능

이러한 권한을 통해 접근한 데이터는 모두 기기 내부에만 저장되며, 외부로 전송되지 않습니다.

4. 결제 정보

  1. 매일읽음의 Premium 결제는 Apple App Store의 In-App Purchase 시스템을 통해 처리됩니다.
  2. 개발자는 이용자의 신용카드 번호, 결제 수단 등 어떠한 결제 관련 정보에도 접근하지 않습니다.
  3. 결제 처리 및 영수증 발행은 전적으로 Apple이 담당하며, 이에 대한 자세한 사항은 Apple Media Services 이용약관을 참조해 주시기 바랍니다.

5. 제3자 제공 및 위탁

개발자는 이용자의 개인정보를 제3자에게 제공하거나 외부에 위탁하지 않습니다. 본 앱은 다음의 외부 서비스를 사용하지 않습니다:

  • 분석 도구(Google Analytics, Firebase Analytics 등)
  • 광고 네트워크
  • 크래시 리포팅 도구
  • 외부 클라우드 백업 서비스
유일한 외부 연동은 Apple의 In-App Purchase(StoreKit 2)이며, 이는 Apple이 직접 처리하는 결제 시스템입니다.

6. 데이터 보관 및 삭제

  1. 이용자가 앱에 입력한 데이터는 이용자가 직접 삭제하거나 앱을 삭제할 때까지 기기 내부에 보관됩니다.
  2. 앱을 삭제하면 모든 데이터가 즉시 영구 삭제되며, 복구할 수 없습니다.
  3. 중요한 데이터는 Premium의 CSV 내보내기 기능을 통해 별도로 백업하실 것을 권장합니다.

7. 아동의 개인정보

매일읽음은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 본 앱은 데이터 수집 자체가 없으므로 모든 연령대에서 안전하게 사용 가능합니다.

8. 이용자의 권리

이용자는 자신의 데이터에 대해 다음의 권리를 가집니다:

  • 열람: 앱 내에서 본인의 모든 데이터를 자유롭게 열람할 수 있습니다.
  • 수정: 앱 내에서 언제든지 본인의 데이터를 수정할 수 있습니다.
  • 삭제: 앱 내에서 개별 데이터를 삭제하거나, 앱 자체를 삭제하여 모든 데이터를 일괄 삭제할 수 있습니다.
  • 이동: Premium의 CSV 내보내기 기능을 통해 본인의 데이터를 외부로 이동할 수 있습니다.
